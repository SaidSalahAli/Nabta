# 📝 نموذج الحلقة المحدّث - Bilingual Episode Form

## التحديثات الجديدة ✅

### 1. **نموذج ثنائي اللغة** (عربي + إنجليزي)

كل حقل اختياري الآن له نسختان:

- **عربي (مطلوب)**: للمحتوى العربي
- **إنجليزي (مطلوب)**: للمحتوى الإنجليزي

### 2. **صورة الغلاف إجبارية** ⭐

- `cover_image` أصبحت **Required**
- يجب أن تكون URL صحيح
- ستظهر على الواجهة الأمامية قبل تشغيل الحلقة

### 3. **حقول جديدة**

- `title_en`: عنوان الحلقة بالإنجليزية
- `short_description_en`: وصف قصير بالإنجليزية
- `description_en`: وصف كامل بالإنجليزية
- `transcript_en`: نص الحلقة بالإنجليزية

## قائمة الحقول الكاملة

### الحقول المطلوبة (\*)

| الحقل                  | النوع   | الوصف                              |
| ---------------------- | ------- | ---------------------------------- |
| `title_ar`             | نص      | عنوان الحلقة (عربي)                |
| `title_en`             | نص      | عنوان الحلقة (إنجليزي)             |
| `category_id`          | اختيار  | التصنيف                            |
| `episode_number`       | رقم     | رقم الحلقة                         |
| `short_description_ar` | نص طويل | وصف قصير (عربي)                    |
| `short_description_en` | نص طويل | وصف قصير (إنجليزي)                 |
| `description_ar`       | نص طويل | وصف كامل (عربي)                    |
| `description_en`       | نص طويل | وصف كامل (إنجليزي)                 |
| `video_url`            | رابط    | رابط الفيديو                       |
| `video_type`           | اختيار  | نوع الفيديو                        |
| `duration_seconds`     | رقم     | المدة بالثواني                     |
| `cover_image`          | رابط    | **صورة الغلاف (مطلوبة دائماً)** ⭐ |

### الحقول الاختيارية

| الحقل             | النوع    | الوصف               |
| ----------------- | -------- | ------------------- |
| `thumbnail_image` | رابط     | الصورة المصغرة      |

| `transcript_en`   | نص طويل  | نص الحلقة (إنجليزي) |
| `sort_order`      | رقم      | ترتيب العرض         |
| `is_published`    | Checkbox | منشورة              |
| `is_featured`     | Checkbox | مميزة               |
| `has_worksheets`  | Checkbox | يوجد أوراق عمل      |

## خطوات ملء النموذج

### 1️⃣ العنوان (بلغتين)

```
عربي:     "مقدمة إلى علم الأحياء"
إنجليزي: "Introduction to Biology"
```

### 2️⃣ اختر التصنيف

```
مثال: "العلوم الطبيعية" (Science)
```

### 3️⃣ رقم الحلقة

```
مثال: 1, 2, 3, ...
```

### 4️⃣ الأوصاف (قصير وكامل بلغتين)

```
الوصف القصير: ملخص سريع (1-2 جملة)
الوصف الكامل: تفاصيل كاملة (عدة فقرات)
```

### 5️⃣ الفيديو

```
الرابط:   https://youtube.com/watch?v=... أو رابط آخر
النوع:    YouTube / Vimeo / MP4 / Stream
المدة:    3600 ثانية (ساعة واحدة)
```

### 6️⃣ الصور (مطلوب cover_image)

```
صورة الغلاف: https://example.com/cover.jpg (إجباري)
صورة صغيرة: https://example.com/thumb.jpg (اختياري)
```

### 7️⃣ النصوص الإضافية (اختياري)

```
نص الحلقة بالعربية والإنجليزية (للمساعدة والترجمة)
```

### 8️⃣ الخيارات

```
☑ منشورة:    هل تريد نشر الحلقة الآن؟
☑ مميزة:     هل هذه حلقة مهمة وظاهرة؟
☑ أوراق عمل:  هل توجد أوراق عمل للحلقة؟
```

## التحقق من الصحة (Validation)

### صور الغلاف

```javascript
// يجب أن تكون رابط صحيح
❌ cover_image: "صورة الغلاف"        // خطأ: ليس رابط
✅ cover_image: "https://..."         // صحيح

// يجب أن تكون موجودة
❌ cover_image: ""                    // خطأ: مطلوب
✅ cover_image: "https://example..."  // صحيح
```

### روابط الفيديو

```javascript
❌ "youtube.com"                      // خطأ: ليس رابط كامل
✅ "https://youtube.com/watch?v=..."  // صحيح
```

### الأرقام

```javascript
❌ episode_number: -5                 // خطأ: سالب
❌ duration_seconds: 0                // خطأ: لا يمكن أن يكون 0
✅ episode_number: 5                  // صحيح
✅ duration_seconds: 3600             // صحيح
```

## ملاحظات مهمة

1. **صورة الغلاف ضرورية** - لن تستطيع إضافة الحلقة بدونها
2. **البيانات ثنائية اللغة** - يجب ملء كل حقل بلغتيه
3. **رابط الفيديو** - تأكد من صحة الرابط قبل الحفظ
4. **الترجمة** - يفضل أن تكون النصوص ترجمة صحيحة وليست جوجل ترانسليت
5. **النقر على حفظ** - سيتم التحقق من جميع الحقول قبل الحفظ
