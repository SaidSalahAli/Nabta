# Nabta Platform - API Integration & Features Setup

## Project Overview

**Nabta** is an educational platform with:

- **Frontend**: React 18 + Vite + Material-UI v6 (RTL/Arabic Support)
- **Backend**: PHP Custom Framework with RESTful API
- **Database**: MySQL (via Composer)
- **Authentication**: JWT-based

## How to Create an Episode

### Step 1: Navigate to Dashboard

```
/dashboard/episodes → Click "Add Episode" button
```

### Step 2: Fill the Form

1. **Title (Arabic)** - Required
2. **Category** - Select from dropdown
3. **Episode Number** - Enter number
4. **Short Description** - Brief summary
5. **Full Description** - Complete details
6. **Video URL** - Link to video
7. **Video Type** - Select type (YouTube, Vimeo, MP4, Stream)
8. **Duration** - Length in seconds
9. **Images** - Thumbnail & Cover (optional)
10. **Checkboxes** - Featured, Published, Has Worksheets

### Step 3: Submit

Click "Save" button to create the episode.

---

## How to Create a Category

### Step 1: Navigate to Categories

```
/dashboard/episode-categories → Click "Add Category" button
```

### Step 2: Fill the Form

1. **Type** - Select: Episodes, Applications, or Worksheets
2. **Name (Arabic)** - Required
3. **Name (English)** - Category name in English
4. **Description (Arabic)** - Arabic description
5. **Description (English)** - English description
6. **Slug** - URL-friendly identifier (auto-fill recommended)
7. **Image** - Category image URL

### Step 3: Submit

Click "Save" button to create the category.

## API Integration Details

### Episode Creation Request

**Endpoint**: `POST /api/v1/admin/episodes`

**Request Body**:

```json
{
  "title_ar": "الرياضيات - الدرس الأول",
  "category_id": 1,
  "episode_number": 1,
  "short_description_ar": "درس تمهيدي",
  "description_ar": "وصف مفصل للدرس",
  "video_url": "https://youtube.com/watch?v=...",
  "video_type": "youtube",
  "duration_seconds": 3600,
  "is_published": true,
  "is_featured": false,
  "has_worksheets": false,
  "sort_order": 1,
  "thumbnail_image": "https://...",
  "cover_image": "https://...",
  "transcript_ar": "نص الحلقة"
}
```

**Response** (201 Created):

```json
{
  "success": true,
  "message": "Episode created successfully",
  "data": {
    "id": 1,
    "title_ar": "...",
    "created_at": "2026-05-14T10:00:00Z"
  }
}
```

---

### Category Creation Request

**Endpoint**: `POST /api/v1/admin/categories`

**Request Body**:

```json
{
  "type": "episodes",
  "name_ar": "الرياضيات",
  "name_en": "Mathematics",
  "description_ar": "دروس الرياضيات",
  "description_en": "Math lessons",
  "slug": "al-riyadiyat",
  "image": "https://..."
}
```

**Response** (201 Created):

```json
{
  "success": true,
  "message": "Category created successfully",
  "data": {
    "id": 1,
    "name_ar": "الرياضيات",
    "type": "episodes"
  }
}
```

## Backend Routes Summary

### Episodes Routes

```php
GET    /api/v1/episodes              - List all episodes
GET    /api/v1/episodes/{id}         - Get episode by ID
POST   /api/v1/admin/episodes        - Create episode (Admin)
PUT    /api/v1/admin/episodes/{id}   - Update episode (Admin)
DELETE /api/v1/admin/episodes/{id}   - Delete episode (Admin)
```

### Categories Routes

```php
GET    /api/v1/categories            - List all categories
GET    /api/v1/categories/{id}       - Get category by ID
POST   /api/v1/admin/categories      - Create category (Admin)
PUT    /api/v1/admin/categories/{id} - Update category (Admin)
DELETE /api/v1/admin/categories/{id} - Delete category (Admin)
```

## Troubleshooting

### Issue: 401 Unauthorized

**Solution**: Check if JWT token is in localStorage. Login again.

### Issue: CORS Error

**Solution**: Verify backend CORS headers are configured correctly.

### Issue: Form validation errors

**Solution**: Check console for detailed error messages. Verify field names match backend requirements.

### Issue: "Slug already exists"

**Solution**: Backend auto-generates slug. Don't provide duplicate slugs.
